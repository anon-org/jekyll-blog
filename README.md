# Jekyll Blog

## Depends on
- Bundle
- Jekyll
- jekyll-paginate-v2
- jekyll-feed

## Getting Started

### Serve
```bash
$ make serve
```

### Build
```bash
$ make build
```

## Deploy your own
- Use template from this repo
- Adjust the configuration at `_config.yaml`
- Add something on `_posts`
- Build
- Deploy!
